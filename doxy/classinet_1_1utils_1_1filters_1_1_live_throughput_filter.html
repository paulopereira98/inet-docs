Detailed Description

Filter that expects a Packet or a packet length and outputs the throughput as double.

Throughput is computed and emitted for the past interval strictly every 0.1s. (To achieve that, this filter employs a timer event.)

For interactive use (with instrument figures, for example), this filter should be preferred to ThroughputFilter.

Recommended interpolation mode: backward sample-hold.
